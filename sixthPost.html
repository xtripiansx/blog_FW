Update

december 7, 2020 - december 13, 2020

This week was very exhausting. But in the end I am proud to finally be able to deploy the application to a Kubernetes cluster.

I mainly followed this tutorial (https://docs.microsoft.com/en-us/azure/aks/tutorial-kubernetes-prepare-app) to begin to understand and use Kubernetes. I created a Azure Container Registry (ACR, it is a private container registry that lets me securely build and deploy the applications and custom code.

In order to do so I had to create a resource group on azure. Using the azure CLI I created my ACR that is linked to my resource group. I tagged the images with my ACR server name and pushed them to the ACR.

Following that I created a kubernetes cluster using the kubernetes CLI. Using a demo project with a kubernetes manifest file, I pushed the demo to my ACR. I had to tag the name of the image with my ACR. This way van I push the manifest file, the image of the demo app will be pushed to my cluster.

After the deployment was done I could navigate to my cluster and see the demo app hosted.

I followed the tutorial numerous times because I did not quite understand how it worked. Eventually after reading some documentation I managed to deploy a manifest file of my app to my cluster.
